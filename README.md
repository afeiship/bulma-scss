# bulma-scss
> Bulma SCSS  The Bulma CSS Framework files converted to SCSS syntax

## sync && rebuild:
```shell
cd .tmp
git clone --depth=1 https://github.com/jgthms/bulma.git
cd ..
gulp sass2scss
```

## before build:
> install ruby sass:(MUST BE RUBY_SASS(NOT Dart/node sass))
```shell
sudo gem install sass
npm install
```
